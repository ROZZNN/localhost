Описание проекта
=================

Общая информация
---------------
Данный проект представляет собой веб-приложение для чтения и управления электронными книгами на базе фреймворка Yii 2. Приложение позволяет пользователям создавать собственную библиотеку электронных книг, загружать, читать и управлять ими.

Версия PHP: >= 7.4.0
Версия Yii2: ~2.0.45

Основной функционал
-----------------
1. Система аутентификации и авторизации:
   - Регистрация новых пользователей
   - Вход в систему (login)
   - Выход из системы (logout)
   - Управление ролями пользователей

2. Управление книгами:
   - Загрузка новых книг в формате .txt
   - Просмотр списка загруженных книг
   - Редактирование информации о книгах
   - Удаление книг из библиотеки
   - Поиск книг по названию и автору

3. Чтение книг:
   - Адаптивное отображение текста под размер экрана
   - Постраничная навигация
   - Сохранение прогресса чтения
   - Автоматическое определение оптимального количества строк на странице

4. Личный кабинет пользователя:
   - Просмотр личной библиотеки
   - Управление загруженными книгами
   - Отслеживание статуса чтения книг (прочитано/не прочитано)

5. Дополнительные функции:
   - Контактная форма для связи с администрацией
   - Система уведомлений о действиях с книгами
   - Защита от несанкционированного доступа к книгам
   - Валидация загружаемых файлов

Структура проекта
----------------
1. assets/ - содержит определения ресурсов (CSS, JavaScript)
2. commands/ - консольные команды (контроллеры)
3. config/ - конфигурационные файлы приложения
4. controllers/ - веб-контроллеры:
   - SiteController.php - основной контроллер сайта
   - AccountController.php - контроллер личного кабинета
   - AdminController.php - контроллер административной панели
5. mail/ - файлы представлений для электронной почты
6. models/ - модели приложения:
   - Book.php - модель для работы с книгами
   - User.php - модель пользователя
   - LoginForm.php - модель формы входа
   - ContactForm.php - модель контактной формы
7. views/ - файлы представлений
8. web/ - точка входа в приложение и веб-ресурсы

Технические особенности
---------------------
1. Файловая система:
   - Книги хранятся в директории web/booksstorage/
   - Каждый файл имеет уникальный идентификатор
   - Поддерживается только текстовый формат (.txt)

2. База данных:
   - Таблица users - информация о пользователях
   - Таблица books - информация о книгах
   - Таблица bookuser - связь между пользователями и книгами
   - Таблица roles - роли пользователей

3. Безопасность:
   - Валидация всех форм
   - Проверка прав доступа к книгам
   - Защита от SQL-инъекций
   - Безопасное хранение паролей
   - Валидация загружаемых файлов

Установка и запуск
----------------
Проект может быть установлен через Composer или из архива. 
Также поддерживается установка через Docker.

Для установки через Composer:
```
composer create-project --prefer-dist yiisoft/yii2-app-basic basic
```

Для запуска через Docker:
```
docker-compose up -d
```

Требования к системе
------------------
- PHP >= 7.4.0
- MySQL >= 5.7
- Composer
- Достаточно места на диске для хранения книг
- Поддержка .txt файлов
- Настроенные права доступа к директории booksstorage

Дополнительная информация
----------------------
Проект разработан с учетом современных требований к веб-приложениям и может быть легко расширен дополнительным функционалом. Основной фокус сделан на удобство чтения и управления электронными книгами.
